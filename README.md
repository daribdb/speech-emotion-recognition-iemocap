# Speech Emotion Recognition — IEMOCAP Dataset

Senior Capstone Project — Nazarbayev University (2024–2026)

## Overview
Speech emotion recognition pipeline trained and tested on the IEMOCAP dataset using 4-class and 6-class emotion categorization.

## Files
- `iemocap_emotion_recognition_csv.ipynb` — main training notebook
- `iemocap_test.ipynb` — testing and evaluation
- `explore_iemocap.py` — dataset exploration
- `filter.py / filter6.py` — data filtering for 4-class and 6-class setups
- `verification_plot.png` — results visualization

## Tech Stack
Python, PyTorch, librosa, NumPy, scikit-learn
